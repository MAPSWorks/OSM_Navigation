Vysvetlivky ku grafu

Tu nájdete vysvetlenie, ako majú byť interpretované grafy, ktoré boli generované programom doxygen.

Uvažujte nasledujúci príklad: 

/*! Neviditelná trieda, ktorá sa v grafe nezobrazuje, pretože došlo k orezaniu grafu. */
class Invisible { };

/*! Trieda, u ktorej došlo k orezaniu grafu. Vzťah dedičnosti je skrytý. */
class Truncated : public Invisible { };

/* Trieda, ktorá nieje dokumentovaná komentármi programu doxygen. */
class Undocumented { };

/*! Odvodená trieda s verejným (public) dedením bázovej triedy. */
class PublicBase : public Truncated { };

/*! Odvodená trieda s chráneným (protected) dedením bázovej triedy. */
/*! A template class */
template<class T> class Templ { };

class ProtectedBase { };

/*! Odvodená trieda s privátnym dedením bázovej triedy. */
class PrivateBase { };

/*! Trieda, ktorá je využívaná triedou Inherited. */
class Used { };

/*! Odvodená trieda, ktorá rôznym spôsobom dedí od viacerých bázových tried. */
class Inherited : public PublicBase,
                  protected ProtectedBase,
                  private PrivateBase,
                  public Undocumented,
                  public Templ<int>
{
  private:
     Used *m_usedClass;
};

Bloky (tj. uzly) v uvedenom grafe majú nasledujúci význam:

  • Čierne vyplnený obdĺžnik reprezentuje štruktúru alebo triedu, pre ktorú bol graf generovaný.
  • Obdĺžnik s čiernym obrysom označuje dokumentovanú štruktúru alebo triedu.
  • Obdĺžnik so šedým obrysom označuje nedokumentovanú štruktúru alebo triedu.
  • Obdĺžnik s červeným obrysom označuje dokumentovanú štruktúru alebo triedu, pre ktorú niesú zobrazené všetky vzťahy dedičnosti alebo obsiahnutia. Graf je orezaný v prípade, kedy ho nieje možné umiestniť do vymedzených hraníc.

Šípky (tj. hrany grafu) majú nasledujúcí význam:

  • Tmavo modrá šípka sa používa pre označenie vzťahu verejnej dedičnosti medzi dvoma triedami.
  • Tmavo zelená šípka označuje vzťah chránenej dedičnosti (protected).
  • Tmavo červená šípka označuje vzťah privátnej dedičnosti.
  • Purpurová šípka kreslená čiarkovane sa používa v prípade, ak je trieda obsiahnutá v inej triede, alebo ak je používaná inou triedou. Je označená identifikátorom jednej alebo viacerých premenných (objektov), cez ktoré je trieda alebo štruktúra zprístupnena.
